# Changelog

## 1.0.2 - 2026-04-20

- 修复一键安装场景下窗口关闭后的异常：后台任务不再因 Flet 会话销毁触发 `An attempt to fetch destroyed session` 连锁报错。
- 优化一键安装统计逻辑：
- 未安装机模不计入失败，单独统计为“未安装”。
- 已安装但云盘无数据的机模，单独汇总为“云盘中无数据”并列出机模名称。
- 增加“最新周期跳过下载”：
- 当本地机模 AIRAC 已等于当前最新周期时，直接跳过下载并显示“已是最新（跳过下载）”。
- 优化 DATA 登录错误提示：
- 当接口返回 `401 invalid credentials` 时，明确提示为 DATA 域名账号/密码错误，避免与 OpenList 账号混淆。
- 版本号升级至 `1.0.2`（客户端默认版本与安装器版本同步更新）。
