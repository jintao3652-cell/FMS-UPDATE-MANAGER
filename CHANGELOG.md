# Changelog

## 1.0.1 - 2026-04-20

- 调整“后备隐藏能源”登录失败提示：`401 invalid credentials` 时明确提示为 DATA 域名账号/密码错误，避免与 OpenList 账号混淆。
- 优化一键安装统计：
- 未安装机模不再计入失败，改为“未安装（数量）”。
- 已安装但云盘无对应数据时，单独汇总为“云盘中无数据（机模列表）”。
- 新增一键安装跳过策略：
- 当检测到本地机模 AIRAC 周期已是最新周期时，跳过下载并在安装状态中标记“已是最新（跳过下载）”。
- 版本号升级至 `1.0.1`（客户端默认版本与安装器版本同步更新）。
